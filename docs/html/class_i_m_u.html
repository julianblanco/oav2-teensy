◆ getmpu6050data()

void IMU::getmpu6050data ( )

This will get data from the mpu6050 at the standard address and then call madgwick. roll, pitch and yaw come from the output of the madgwick filter. It is called by getIMUdata in the imu sampling loop.

This function updates the roll, pitch, yaw variables in the class object
